



# 🍔 Food_Cart

Modern ve kullanıcı dostu bir yemek sipariş uygulaması.
React ekosistemi kullanılarak geliştirilmiş, restoranlara göre ürün listeleme ve sepet yönetimi özelliklerine sahiptir.

---

## 🚀 Özellikler

* 🏪 Restoran listeleme
* 🍽️ Restorana özel ürünleri görüntüleme
* 🔍 Dinamik route yapısı (`/restaurant/:id`)
* 🛒 Sepete ürün ekleme / çıkarma
* 🔢 Ürün miktarı güncelleme
* 💾 Redux ile global state yönetimi
* 🔔 Bildirim sistemi (React Toastify)
* ⚡ Hızlı ve performanslı veri çekme (Axios)
* 🎨 Modern UI (Tailwind CSS)
* 🧩 Mock API (json-server)
